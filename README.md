opjay2 is a indexer of Go/C/Cpp lanuages written on Go lang. opjay2 create "./tags" file and you can find any name you need, for example with 
"gvim -t <name>" command.
Tested on linux-5 and last Go sources.
Simple run 
$ opjay2 [dirname]
